# Clustering_Analysis_Customers_Segmentation_Project

In this project, customers are required to be segmented according to the purchasing history obtained from the membership cards of a big mall.
<br>
<br>
After getting to be known the data set quickly, performed Exploratory Data Analysis. Observed the distribution of customers according to different variables, and also discovered relationships and correlations between variables. 
<br>
<br>
Finally, clustered customers using the <b>K-Means Clustering method</b>, and after that label the clusters.
